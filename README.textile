h2. Running

odkmaker is built on the Ruby Rack server stack, so look up how to run Rack servers, and apply that to config.ru. Phusion Passenger works.

For a quickstart type local dev run, make sure you have all the required stuff, as well as a webserver gem such as mongrel, and then type @shotgun config.ru@.
