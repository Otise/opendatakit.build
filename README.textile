ODK Build is built on the Ruby Rack server stack. Go to "http://build.opendatakit.org":http://build.opendatakit.org to give ODK Build a spin; unless you wish to write code for Build, this should be your best option. If you would like to use it locally on your own machine, try the following steps.

h2. Install Dependencies
* "Ruby 1.8.6+":http://www.ruby-lang.org/en/downloads
* "Tokyo Cabinet":http://1978th.net/tokyocabinet/spex-en.html#installation (from source)
* "RubyGems":http://docs.rubygems.org/read/chapter/3

You will need to install Tokyo Cabinet from source. If you are in Windows, you will need "Cygwin":http://cygwin.com/. If you are on a Mac, you will need "XCode":http://developer.apple.com/technologies/xcode.html. If you are on Linux, you already know what to do.  

h2. Install Gems

* ffi
* rufus-tokyo
* rack
* json
* warden
* shotgun
* mongrel
* sinatra 0.9.6

You can install all with @gem install ffi rufus-tokyo rack json warden shotgun mongrel sinatra --version "0.9.6"@

h2. Run Server

Check out the source code and in the base directory, you will find _config.ru_ and _odkbuild_server.rb_. In that folder, run @shotgun config.ru@.

Point your browser to "http://localhost:[port]":http://localhost:9393 (where port is probably 9393) to start using the application.
